# SALAM Restaurant Website

## Project Overview
This is a multi-page responsive website for the "SALAM" restaurant, inspired by modern premium restaurant sites. The project demonstrates semantic HTML5 structure and organized CSS architecture.

## Pages
- **Home** (index.html): Hero block with restaurant introduction
- **About Us** (pages/about.html): Description of the concept and chef
- **Menu** (pages/menu.html): Dish cards with images, names, descriptions, and prices
- **Contacts** (pages/contacts.html): Contact information and embedded Google Map

## Technologies Used
- HTML5 with semantic tags (header, nav, main, section, article, footer)
- CSS3 with Flexbox and Grid layouts
- Responsive design with media queries

## CSS Architecture
- **reset.css**: Resets default browser styles
- **base.css**: Base styles, typography, and color variables
- **layout.css**: Layout styles using Flexbox and Grid
- **components.css**: Component styles for buttons, cards, navigation, etc.

## Features
- Modern minimalist design
- Large hero block with background image and overlay
- Rounded buttons with hover effects
- Dish cards with hover animations
- Consistent navigation with active state highlighting
- Fully responsive across devices
- No JavaScript or external frameworks used

## Setup
1. Clone or download the project files.
2. Open index.html in a web browser to view the site.
3. Replace placeholder images in the `images/` folder with actual restaurant photos.

## Note
This is an educational project demonstrating clean, semantic code and modern CSS techniques. All content is fictional and for demonstration purposes only.
